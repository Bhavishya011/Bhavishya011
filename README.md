[![Repo](https://img.shields.io/badge/GitHub-UAV--DRL-0D1B4B?style=for-the-badge&logo=github)](https://github.com/Bhavishya011/UAV-DRL)

![Gymnasium](https://img.shields.io/badge/Gymnasium-0081CB?style=flat-square&logo=openai&logoColor=white)

> **0% collision rate from 100% baseline | 85% faster convergence**

A 3D UAV navigation policy trained end-to-end using **Deep Q-Networks** in obstacle-dense environments, comparing three DQN variants with systematic reward engineering.

| Variant | Convergence | Collision Rate |
|:---|:---:|:---:|
| Baseline DQN | — | 100% |
| Double DQN | ↑ faster | ↓ lower |
| **Dueling DQN** | **85% faster** ✅ | **0%** ✅ |

- Custom **reward shaping** to eliminate reward exploitation
- **Replay buffers + target networks** for training stability
- Diagnosed failure modes via **Q-value diagnostics**
